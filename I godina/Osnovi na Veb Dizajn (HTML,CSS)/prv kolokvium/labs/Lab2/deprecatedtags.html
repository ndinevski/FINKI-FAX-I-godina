  2. <basefont>

  3. <center>
    • Specifies centered text
  4. <dir>
    • Specifies a directory list
  1. <plaintext>
    • Specifies plaintext
  2. <strike>
    • Specifies strikethrough text
  3. <u>
    • Specifies underlined text
